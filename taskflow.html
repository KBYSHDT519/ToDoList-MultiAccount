Google API 設定

設定方法（クリックで展開）
  1. Google Cloud Console を個人Gmailで開く（組織なしのプロジェクト）
  2. Google Tasks API を有効化
  3. 認証情報 → OAuthクライアントID → ウェブアプリ → http://localhost:8080 を追加
  4. OAuth同意画面 → 外部 → テストユーザーに自分を追加
  5. クライアントIDを上に貼り付け → 保存

このアプリは http://localhost:8080/taskflow.html で開いてください

サーバー起動: cd ファイルのフォルダ → python3 -m http.server 8080
